# Smart Fridge — GUI skeleton

Скелет веб-приложения по макетам Ксюши. Кнопки пока заглушки.

## Запуск
```bash
pip install flask
python app.py
```
Откроется на http://127.0.0.1:5000

## Страницы
- `/` — главная (информация о проекте)
- `/vr-shell` — список продуктов из холодильника. У скоропортящихся включается красная подсветка через CSS-переменную `--urgency` (0..1), градиент от ярко-красного к прозрачному.
- `/ai-assistant` — AI-ассистент: «Давайте поскорее приготовим», «Лучшее на сегодня», «ПП-рецепты»
- `/recipe` — детальная страница рецепта (тип блюда, время, продукты, текст)
- `/statistics` — заглушка под страницу статистики

## Стек
- Flask (Python) — бэк
- Jinja2 шаблоны + один CSS файл — фронт. Без сборки, без npm.

## Что дальше
- Подсветка urgency сейчас считается по индексу в моке (`app.py`). Когда появится бэк — заменить `VR_PRODUCTS` на реальные данные с полем `urgency` (0..1) или сроком годности, из которого его пересчитать.
- Кнопки «Начать», «Попробовать», «Перейти», «Узнать больше», «Начать готовить», «Log out» — пустышки, ждут логику.
- Картинки-плейсхолдеры (серые квадраты) заменить на реальные фото продуктов/блюд.
